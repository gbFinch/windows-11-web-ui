# Windows 11 Web UI

A high-fidelity web recreation of the Windows 11 desktop environment built with React, TypeScript, and Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run test` — Run tests
- `npm run lint` — Lint source code
- `npm run format` — Format source code
